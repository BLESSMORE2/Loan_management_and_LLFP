1. Forward Exposure Methodology

Overview:This method calculates Expected Credit Loss (ECL) by estimating the lender’s potential losses using Exposure at Default (EAD) at each point in time. For each date, the EAD is used to calculate the probability of default (PD) and the expected loss if the borrower defaults (LGD). These risks are aggregated for each time period, and the total ECL is determined by discounting these values using the Effective Interest Rate (EIR). The method calculates both 12-month ECL (covering the next 12 months) and Lifetime ECL (covering the entire life of the loan).

2. Cash Flow Calculations Methodology

Overview: In this approach, projected cash flows at each point in time are adjusted to account for the borrower’s probability of default (cumulative PD) and the potential loss in case of default (LGD). The 12-month ECL represents the expected loss for the full loan term, but the PD applied is only the cumulative PD up to 12 months. The Lifetime ECL considers the full term of the loan, using cumulative PD until maturity. All losses are discounted to their present value using the Effective Interest Rate (EIR) to reflect the time value of money.

3. Simple EAD Methodology

Overview: This method simplifies the calculation by focusing on the loan’s current balance (Exposure at Default, or EAD). The probability of default (PD) and potential loss (LGD) are applied directly to the loan’s balance. For 12-month ECL, the 12-month PD is used to calculate the expected loss for the next 12 months. For Lifetime ECL, the lifetime PD is used to estimate the loss over the full term of the loan. All values are discounted using the Effective Interest Rate (EIR) to reflect the time value of money. This approach is quick and effective, especially when detailed cash flow projections aren’t available.

Effective Interest Rate (EIR) in All Methods

In all methods, the Effective Interest Rate (EIR) is used to discount future losses to their present value. The Forward Exposure Methodology uses exposure at default for each point in time, while the Cash Flow Calculations Methodology uses projected cash flows for each point in time. For the Simple EAD Methodology, 12-month ECL uses 12-month PD, and lifetime ECL uses lifetime PD.
